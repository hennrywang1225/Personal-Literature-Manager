## 下载

Windows 用户请前往 Releases 页面下载最新版 `.exe` 安装包：

https://github.com/hennrywang1225/Personal-Literature-Manager/releases

- 本地文献管理
- 文献分类
- 阅读界面
- 不需要联网
- 不需要账号
